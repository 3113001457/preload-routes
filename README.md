## Project setup

### Compiles and hot-reloads for development

```bash
# install dependencies
yarn

# development serve
yarn serve

# open: http://localhost:10240/
```

### Compiles and minifies for production

```bash
# production build
yarn build

# production build with example
yarn build:example
```
